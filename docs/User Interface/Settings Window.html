AErender Launcher Help User Interface

Settings Window

All essential Launcher settings are present in this window.

  • After Effects render engine path - is a required field for Launcher to work properly, because all executables will be passed to aerender.
  • Style - if you want, you can change Launcher visual style from available ones. If you want, you can creat your own style using RAD Studio FireMonkey style designer (but please, remember to create as much vector style as possible), and submit it via Pool Request on AErender Launcher GitHub page.
  • Default projects directory - this path will be passed to Project Open Dialog for easier and faster access to your project files directory.
  • Default output directory - this path will be passed to Output Save Dialog for easier and faster access to your output files directory.
  • On Render Start - will determine what Launcher will do on render start. Note that Close Launcher parameter will not work, if aerender Handle is enabled.
  • Handle aerender processes inside Launcher - or aerender Handle is used to enable rendering progress display directly inside AErender Launcher.
  • Delete temporary files on close - Launcher creates .log and .bat / .command files. This option will enable deletion of this files each time Launcher closes.

Holding Alt / Option key, will display hidden Reset button. Reset Launcher - will remove Launcher configuration and all the created by Launcher files.
