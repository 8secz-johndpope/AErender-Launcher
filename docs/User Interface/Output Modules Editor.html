AErender Launcher Help User Interface

Output Modules Editor

Output Modules Editor is used to create, edit and delete aerender Output Modules, that Launcher can use. If you created any custom After Effects output modules, you must add them here in order to use them with Launcher.

  • Output Modules list - all names of your output modules are listed there. By default there are only After Effects default modules present. You can add(+) and delete(-) any of the present output modules.
  • Preset name - shows how your output module will be called inside Launcher. Please note, that is not the output module itself.
  • Output module - After Effects output module name. Please note, that provided output module MUST EXIST in After Effects.
  • Output file name structure - is a mask for aerender output file. aerender will automatically fill specific values into that mask. For easier editing, all available masks values are suggested in the tabs under the edit field. All the values splited by categories exactly like in After Effects. You can also drag them onto the field.
    Example: Output Module: Photoshop
    Predefined mask: [compName]_[#####].[fileextension]
    Output name: Comp 1_00001.psd
