AErender Launcher Help Introduction

After Effects Render Engine

As said on Adobe After Effects Official Support page aerender is a program with a command-line interface with which you can automate rendering. The executable file is located in the same folder as the primary After Effects application.

The whole AErender Launcher is built on utilizing features of that executable. It generates batch/command files in a temporary directory and executes it.

When using Launcher, you must provide path to aerender. However, latest versions of Launcher are able to detect automatically latest version of After Effects Render Engine.
